Подготовка фидов для Insales и СберМегаМаркет

  • Okeandra - обработка 1СItems.xls: установка безопасного остатка.
  • Сбер - Duty Free - подготовка фида для Duty Free из наличия на Площади
  • СберМегаМаркет - подготовка фида: установка срока "день в день" для избранных товаров.
  • Отгрузка со Сбера - подготовка фида для отгрузки со склада Сбера.

    • Файлы

    Выгрузка из 1С -> 1CItems.xls
    Выгрузка из 1С + лимиты -> 1CItems_feed.xls

    Insales фид (основные товары)-> 79749.xml
    Insales фид (DutyFree) -> 84930.xml
    Фид для СберМегаМаркета -> resultDeliveryFromOkeandra.xml
    Фид для СберМегаМаркета (Duty Free) -> SberDutyFree.xml
    Товары отгружаемые "день в день" -> dayperday.txt

    Фид для товаров отгружаемых со склада Сбера -> xml
    Список товаров отгружаемых со склада Сбера -> SberItems.txt
